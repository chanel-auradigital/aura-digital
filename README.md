# Aura Digital

Proyecto de marca personal y marketing digital.

## Estructura

- **prompts/** — Plantillas de prompts para Claude (reutilizables, con placeholders)
- **clientes/** — Un folder por cliente con su brief completado y outputs generados
- **branding/** — Identidad visual, naming, tono de voz y guías de marca
- **estrategia-marketing/** — Estrategias y planes generados (outputs)
- **contenido/** — Calendario editorial, copies, guiones y artículos (outputs)
- **redes-sociales/** — Gestión de canales, publicaciones y comunidad (outputs)
- **assets/** — Recursos gráficos, imágenes, logos y plantillas

## Flujo de trabajo

1. Tomar un prompt de `prompts/`
2. Completar los `[PLACEHOLDERS]` con los datos del cliente
3. Guardar el brief completado en `clientes/nombre-cliente/`
4. Guardar los outputs generados por Claude en la carpeta temática correspondiente
