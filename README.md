# Aura Digital

Proyecto de marca personal y marketing digital.

## Subproyectos

- **branding/** — Identidad visual, naming, tono de voz y guías de marca
- **estrategia-marketing/** — Planificación, objetivos, KPIs y campañas
- **contenido/** — Calendario editorial, copies, guiones y artículos
- **redes-sociales/** — Gestión de canales, publicaciones y comunidad
- **assets/** — Recursos gráficos, imágenes, logos y plantillas
